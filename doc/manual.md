## Hardware Angeschlossen 

Als ersters besorgen Sie sich einen Ardoino Mega mit einem extra Panel. Zudem müssen Sie sich noch zwei Knöpfe mit eingebauten LEDs kaufen. 

Laden Sie zusätzlich den Ardoino Code der dazu gefügt wurde herunter. Wie auch das HTML Dokumment um das Ergebniss auf einer Webseite Anzeigen zu können. 

Bei dem Ardoino müssen Sie die beiden Knöpfe bei D7 und bei D5 anschließen. Zusätzlich müssen Sie den Ardoino noch an ihr Endgerät anschließen.
